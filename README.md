# ⚡ Electric Vehicle Dashboard (Power BI)

## 📌 Overview
This project presents an interactive Electric Vehicle (EV) Dashboard built using Power BI to analyze trends and insights related to electric vehicles. It provides a comprehensive view of EV adoption, vehicle distribution, and performance metrics across different regions and manufacturers.

---

## 🚀 Features
- 🚗 Total Vehicles Analysis
- 📊 BEV vs PHEV Distribution
- 📈 Vehicles Trend by Model Year
- 🌍 State-wise Vehicle Distribution (Map View)
- 🏭 Vehicles by Manufacturer (Tesla, Nissan, etc.)
- ⚡ CAFV Eligibility Analysis
- 🎛️ Interactive Filters (City, Electric Utility, Vehicle Type)
